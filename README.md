## Finance-Tracker-Chat-Bot
Finance Tracker — AI-Powered Personal Finance Assistant This is a conversational personal finance management tool built with Python, powered by an OpenAI GPT model and a local SQLite database. Users interact with it through a Gradio chat interface entirely in natural language — no forms, no buttons, just conversation.

### Key Functions - 
Salary tracking — log your monthly income for any month and year
Expense logging — record spending with amount, category, description, and date
Balance check — instantly see how much is left after expenses
Spending summary — get a category-wise breakdown of monthly expenses with percentages
